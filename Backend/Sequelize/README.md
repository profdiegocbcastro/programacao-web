# API Simples em Camadas com Sequelize e Swagger – Exemplo Educacional

Este projeto foi criado para demonstrar o Sequelize junto com a documentação de APIs com Swagger.

## Objetivo do Exemplo

Este exemplo tem como objetivo ensinar:
- O uso do Sequelize como ORM
- A definição de models e relacionamentos

## Persistência de Dados com Sequelize

Neste projeto, os dados são persistidos em banco de dados relacional utilizando o Sequelize, um ORM completo para Node.js.

O Sequelize permite:
- Definição de models
- Criação de relacionamentos (associations)
- Operações de CRUD sem escrita direta de SQL
- Maior abstração e produtividade no acesso aos dados

Essa abordagem é ideal para projetos que exigem:
- Padronização
- Forte modelagem de domínio
- Facilidade de manutenção

## Conclusão

Este exemplo tem caráter educacional, demonstrando como utilizar um ORM completo em conjunto com boas práticas de arquitetura e documentação. 
Ele serve como base para aplicações backend mais robustas utilizando Express, Sequelize e Swagger.
