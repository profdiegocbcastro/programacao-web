# API Simples em Camadas com TypeORM e Swagger – Exemplo Educacional

Este projeto é uma API criada para demonstrar a arquitetura em camadas utilizando o TypeORM junto com o Swagger.

## Objetivo do Exemplo

Este exemplo tem como objetivo ensinar:

- O uso do TypeORM como ORM
- A definição de entities e relacionamentos

## Persistência de Dados com TypeORM

Neste projeto, os dados são persistidos em banco de dados relacional utilizando o TypeORM, um ORM que trabalha com forte uso de decorators e classes.

O TypeORM permite:
- Definição de Entities
- Mapeamento direto entre classes e tabelas
- Criação de relacionamentos (OneToMany, ManyToOne, ManyToMany)
- Operações de CRUD com alto nível de abstração

Essa abordagem é amplamente utilizada em projetos que priorizam:
- Organização
- Tipagem forte
- Clareza no modelo de domínio

## Conclusão

Este exemplo tem caráter educacional, demonstrando o uso de um ORM moderno e amplamente utilizado em conjunto com boas práticas de arquitetura e documentação.
Ele serve como base para aplicações backend mais robustas utilizando Express, TypeORM e Swagger.
